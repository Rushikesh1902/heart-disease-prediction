# Heart Disease Prediction

This repository contains a machine learning project that predicts the presence of heart disease using clinical data. The project includes data preprocessing, feature analysis, model training, model evaluation, and performance comparison.

This project was completed as part of the **Pluto Academy AI & ML Internship Program**.

## Project Objectives

- Load and explore the dataset
- Preprocess the data
- Perform feature analysis
- Train and compare multiple machine learning models
- Identify the best performing model

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset Source

Heart Disease Dataset:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset
